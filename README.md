Nombre del proyecto: TerminalSearch - Buscador de archivos y carpetas a través de la terminal de Linux (Ubuntu)

Descripción:
Este script permite buscar archivos y carpetas (directorios) en el sistema que coincidan con un patrón de búsqueda especificado por el usuario.

Funcionalidades:
    1. Permite buscar archivos y carpetas.
    2. Valida el patrón de búsqueda y la ruta de búsqueda.
    3. Muestra la ruta completa del archivo o carpeta encontrado.
    4. Ofrece dos tipos de búsqueda: simplificada y extendida.
    5. Maneja errores comunes.

Argumentos:
Patrón de búsqueda: El string que se desea buscar.

Retorno:
Muestra en pantalla la ruta (path) del archivo o directorio encontrado.

Requisitos: Python 3

Cómo usar:
Ejecute el script: python TerminalSearch.py

Ingrese patrón de búsqueda: *.py
Ingrese objeto a buscar: 1-Archivo, 2-Directorio: 1
Ingrese ruta de búsqueda(path)...

           Ingrese [.] para una búsqueda local
